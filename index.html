So sánh đoạn văn bản

Dán đoạn văn bản cũ và đoạn tương ứng ở văn bản mới để đối chiếu từng câu từ — không cần 2 văn bản có cấu trúc giống nhau.

Sử dụng trên Điện thoại

Kết nối điện thoại và máy tính vào cùng mạng Wi-Fi, sau đó quét mã QR bên dưới để mở ứng dụng trên điện thoại:

HOẶC TRUY CẬP ĐƯỜNG DẪN:
http://192.168.1.x:8090
💡 Mẹo bảo mật & tiện lợi: Nếu bạn tải mã nguồn này lên GitHub Pages, hãy đăng nhập cùng tài khoản Google Drive trên điện thoại và máy tính để tự động đồng bộ mọi lúc mọi nơi mà không cần mở máy tính!

Cấu hình Google Cloud API

Ứng dụng đã cấu hình sẵn Client ID để chạy ngay trên http://localhost:8090. Nếu bạn chạy trên GitHub Pages hoặc máy chủ riêng, bạn cần lên Google Cloud Console tạo OAuth Client ID cho Web Application và dán vào đây để lưu trữ.
